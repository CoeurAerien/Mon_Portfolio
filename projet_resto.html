Ojectif :

L’objectif de ce projet est de découvrir et mettre en pratique le modèle MVC (Modèle-Vue-Contrôleur) en PHP, tout en apprenant à interagir avec une base de données MySQL. Le projet consiste à créer un site web de critiques de restaurants, comme Tripadvisor ou LaFourchette, qui permet aux utilisateurs de consulter, noter et commenter des restaurants.

Outils :Utilisation du serveur installé en local Wampserver, de la base de donnée PHPMyAdmin, de l'éditeur de code VisualStudioCode

Synthèse :

Les visiteurs peuvent :

  • rechercher un restaurant par plusieurs critères,
  • consulter les informations d’un restaurant,
  • lire les avis des autres utilisateurs,
  • s’inscrire sur le site.

Les utilisateurs connectés peuvent :

  • modifier leur profil,
  • indiquer leurs types de cuisine préférés,
  • "aimer" des restaurants,
  • noter et commenter des restaurants.

Chaque restaurant a des photos, une note de 1 à 5, et des commentaires. Le site prend en compte les préférences des utilisateurs pour personnaliser les résultats de recherche.

Contexte :

Le site s’appelle r3st0.fr et vise à aider les utilisateurs à trouver un restaurant en fonction de plusieurs critères (nom, adresse, type de cuisine, etc.). Il stocke des informations sur les restaurants, les avis des utilisateurs, et les préférences culinaires. Deux types d’utilisateurs utilisent le site :

  • les visiteurs non connectés
  • les utilisateurs enregistrés (authentifiés)
Étape :

Partie 1 : Découverte du MVC en PHP pour structurer proprement le code.

Partie 2 : Création de la base de données MySQL (tables restaurants, utilisateurs, critiques, types de cuisine...).

Partie 3 : Développement de l’interface utilisateur (HTML/PHP)

  • Page d’accueil,
  • Fiches de restaurant,
  • Formulaires d’inscription, de connexion, de recherche.

Partie 4 : Ajout des fonctionnalités

  • Système de notation et commentaires,
  • Affichage des restaurants préférés,
  • Gestion du profil utilisateur.

Partie 5 : Développement de l’interface utilisateur (HTML/PHP)

  • Affichage des meilleurs restaurants,
  • Administration des avis,
  • Recherche améliorée selon les goûts personnels.
