Objectif : Création d'un site web pour un groupe de photographes
Affichage des photos des différents photographes
Mise en place d'un formulaire d'inscription et de connexion pour les membres

Outils : Utilisation du serveur local WampServer, de la base de données PHPMyAdmin, de l’éditeur de code Visual Studio Code

Synthèse :
  • Création des différentes pages du site
  • Implantation de la base de données
  • Révision du code
  • Tests des différentes fonctionnalités
Contexte :

Le site du Club Photo de Limoges.
Il s'agit de mettre en place une interface permettant de gérer les articles et les différentes photos proposées par les membres du club.

Étapes :

Partie 1 : Structure du site
Créer l'arborescence du site dans l'explorateur de fichiers et l'ouvrir dans VS Code

Partie 2 : Page d'accueil
Créer la page d’index et y inclure les différents blocs via une méthode PHP

Partie 3 : Réalisation

  • 3.1 : Gestion des photos : afficher toutes les photos selon le type d'article, voir les détails d'une photo
  • 3.2 : Affichage des articles triés par date de parution, affichage des photos associées à chaque article, ajout de photos à un article
